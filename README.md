# 📌 RESUMO DO APLICATIVO

O **V7 Security System** é uma suíte avançada de monitoramento por vídeo baseada em:

- 🧠 Inteligência Artificial
- 👁️ Visão Computacional
- 🔥 Detecção Inteligente de Eventos
- 📡 Automação de Alertas
- 🎯 Rastreamento em Tempo Real

O sistema captura janelas específicas de softwares de câmeras (Yoosee, DVR, NVR, CMS, etc), processa os quadros em tempo real e identifica eventos críticos automaticamente.

# ⚙️ COMO USAR

```bash
1. Execute config_gui.py

2. Selecione a janela da câmera

3. Configure:
   - Sensibilidade
   - IA
   - Telegram
   - Zonas de risco

4. Ative o monitoramento

5. Os vídeos serão salvos em:
   /gravacoes

6. Alertas serão enviados automaticamente
